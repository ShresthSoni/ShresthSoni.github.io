In this challenge we see another file with setuid permissions. When we run it, a password is asked. Then we use 'ltrace' command to see the library called by the file. There we see that it is comparing our enteres string with another string. Thst is the password.

Now we have gotten into a new shell. From here we can 'cat' the password for next level.
